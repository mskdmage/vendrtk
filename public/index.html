WHAT IS THIS?
-------------
  vendrtk ingests vendor invoices and statements of work (PDF),
  runs them through a parsing pipeline, and returns structured
  output suitable for CSV export.

QUICK START
-----------
  1. Use [Browse] below to select a .PDF document
  2. Press [UPLOAD] to transmit the file to the server
  3. Review the response data shown under OUTPUT

SYSTEM REQUIREMENTS
-------------------
  * PDF input files only
  * Web browser with JavaScript enabled
